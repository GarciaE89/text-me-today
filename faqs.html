Text-Me-Today

Daily Texts Made Easy

This site was made as a special project via the efforts of John Woodland, Emmanuel Garcia, Riley Smith and Jaxson Connors. It uses a combination of HTML, CSS with Bootflat. Javascript and several APIs including Openweathermap, for Weather, Api.icndb.com for Jokes, Dulce-affirmations-api for daily affirmations and Textbelt.com for the texts sent out.

This was a project of love and we would welcome any feedback or questions.
